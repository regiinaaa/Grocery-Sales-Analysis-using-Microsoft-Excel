# Grocery-Sales-Analysis-using-Microsoft-Excel
## Project Overview
This project analyzes retail grocery sales data using Microsoft Excel. The project demonstrates an end-to-end analytics workflow including data validation, data transformation, Excel functions, Pivot Tables, and dashboard development to support business decision-making.
## Business Problem
Retail businesses need to monitor sales performance across products and categories. Without effective reporting, it is difficult to identify top-performing products, sales trends, and customer purchasing patterns. This project provides an interactive Excel dashboard to support business monitoring.
## Objectives
- Validate sales data
- Transform raw data into analytical data
- Build Pivot Tables
- Create an interactive dashboard
- Generate business insights
